## Usage
To create a button:

Add the `cta` class to your link.
Add a second class to choose the color: `cta-red`, `cta-green`, `cta-blue`, or `cta-yellow`.
Add an icon using classes: `icon-` plus:
'check'
'doc'
'download'
'email'
'go'
'more'
'search'
I used an icon font from Fontello.
